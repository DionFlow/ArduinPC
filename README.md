### Hello.

To create ArduinPC you need Ardino Nano and 1.8" TFT ST7735.

### Library:
https://github.com/adafruit/Adafruit-ST7735-RESLibrary


### Pins:
- X Button - D4;
- Y Button - D7;
- Z Button - D2;
- TFT LED - 3v3;
- TFT SCK - D13;
- TFT SDA - D11;
- TFT AO - D9;
- TFT RESET - D8;
- TFT CS - D10;
- TFT GND - GND;
- TFT VCC- 5V;

code for project you can find in Code
